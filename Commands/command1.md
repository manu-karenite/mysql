## 1. To Clear the Terminal

Ctrl + L

## 2. To create a new Database

CREATE DATABASE <DATABASE_NAME>;

## 3. Show all DATABASES

SHOW DATABASES

## 4. USE A DATABASE

USE <DATABASE_NAME>

## 5. DROP A TABLE

DROP TABLE <TABLE_NAME>

## 6. Read Commands from File

SOURCE <file_path>/<name_of_file>.sql
